ColorChanger

ColorChanger is a simple web project that lets you change the background color of a webpage with a click. Perfect for beginners learning HTML, CSS, and JavaScript!

How to Use

Clone the repository.

Open index.html in your browser.

Click the button to change the background color!

Project Structure
colorchanger/
├─ index.html      # Main HTML file
├─script.js       # JavaScript for color changes
├─style.css       # Styling
└─ README.md       # Project info
